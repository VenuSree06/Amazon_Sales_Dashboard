## Overview

The Amazon Sales Dashboard is an interactive data analytics project developed using Python and Streamlit.

This dashboard analyzes Amazon product data and provides valuable business insights through interactive charts, KPI cards, and filtering options.

The project demonstrates data cleaning, exploratory data analysis (EDA), business intelligence reporting, and dashboard development skills.

---

## Objectives

The main objectives of this project are:

- Analyze Amazon product data
- Identify product rating trends
- Understand discount strategies
- Explore category-wise performance
- Build an interactive business dashboard
- Present insights in a professional format

---

## Business Questions Answered

### 1. What is the average discount percentage offered on Amazon products?

Helps understand overall discount strategies used across products.

### 2. Which product categories contain the highest number of products?

Identifies dominant categories on the platform.

### 3. What is the average product rating?

Measures overall customer satisfaction.

### 4. How are product ratings distributed?

Analyzes the spread of customer ratings.

### 5. Which categories receive the highest discounts?

Identifies categories with aggressive pricing strategies.

### 6. Is there a relationship between price and rating?

Examines whether expensive products receive better ratings.

### 7. What pricing patterns exist across products?

Provides insights into product pricing behavior.
